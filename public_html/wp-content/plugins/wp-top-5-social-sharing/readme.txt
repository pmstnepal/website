WP Top 5 Social Sharing is a easy to use and flexible WordPress plugin. This plugin add a floating icon set for sharing your blog posts, pages, attachments etc on top 5 social media (Facebook, Twitter, Google Plus, LinkedIn & Digg).

== Description ==
WP Top 5 Social Sharing is a easy to use and flexible WordPress plugin. This plugin add a floating icon set for sharing your blog posts, pages, attachments etc on top 5 social media (Facebook, Twitter, Google Plus, LinkedIn & Digg).

https://www.youtube.com/watch?v=kg48V-EP6DQ

== Installation ==
1. Upload `wp-top-5-social-sharing` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
